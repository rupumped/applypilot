Privacy Policy

Last Updated: March 25, 2026

Self-hosted software: ApplyPilot is open-source software that you run on your own computer or server. All your data stays in your own database — it is never sent to or stored by the project authors. The person or organisation who deployed this instance is the data controller for any personal data stored here.

1. Information Stored

1.1 Information You Provide

  • Account Information: Email address, name, and password when you create an account
  • Profile Data: Professional information including work experience, education, skills, and career preferences — entered manually or extracted from a resume you upload
  • Job Application Data: Job postings and job descriptions you submit for analysis
  • API Keys: If you use Bring Your Own Key (BYOK) mode, your API key is stored encrypted in the local database

1.2 Log Data

  • Server Logs: IP addresses, access times, and pages viewed — written to local log files on the server running this instance, retained for up to 30 days

1.3 Information from Third Parties

  • Google OAuth: If you sign in with Google, only your email address and name are stored locally
  • Chrome Extension: When you choose Analyze This Job, posting content from that page is sent to this instance. When you choose Match Form To Profile, visible form field data and your profile are sent to this instance for suggestions. Nothing is sent until you click one of those actions.

2. How Your Information Is Used

Your information is used solely to:

  • Analyze job postings, match your profile against job requirements, and generate personalized application materials
  • Keep you authenticated across sessions
  • Send email verification and password reset messages (only if SMTP is configured on this instance)

Your data never leaves this instance except when job descriptions and profile data are sent to the Google Gemini API to generate AI output (see Section 4). No data is sent to the project authors.

3. Data Storage and Security

All data is stored in the PostgreSQL database running alongside this instance:

  • API Keys: Encrypted using Fernet symmetric encryption before being stored
  • Passwords: Hashed using bcrypt — never stored in plain text
  • Authentication tokens: Short-lived JWTs stored in your browser's localStorage; revocation list stored in Redis
  • Security: The instance operator is responsible for securing the host, database, and network

4. Third-Party Services

Depending on how this instance is configured, the following external services may be contacted:

  • Google Gemini API: Job descriptions and your profile data are sent to Google's Gemini API to generate AI analysis. This is governed by Google's Privacy Policy. The instance operator's server API key or your own BYOK key is used.
  • Google OAuth: If Google sign-in is enabled on this instance, authentication is handled by Google's OAuth service. Only your email address and name are stored locally.
  • Analytics (optional): If PostHog analytics are enabled by the instance operator, usage events are sent to PostHog only with your explicit consent via cookie preferences. This is disabled by default in the self-hosted configuration.

5. Your Rights

Because all data is stored locally on your instance, you have direct control:

5.1 Access and Export

Use the "Export My Data" feature in Dashboard → Settings to download everything stored about you in JSON format.

5.2 Correction

Update or correct your personal information at any time through your profile settings.

5.3 Deletion

Use the "Delete Account" feature in Dashboard → Settings to permanently and immediately delete your account and all associated data. This action cannot be undone.

5.4 Data Portability

Your data export is in machine-readable JSON format, usable by other systems.

6. Cookies and Local Storage

This application uses browser storage for the following purposes:

  • Authentication: Your login token is stored in localStorage to keep you signed in. Required for the application to function.
  • Cookie Consent: Your cookie preferences are stored in localStorage so the consent banner does not reappear on every visit.
  • Analytics (if enabled): PostHog uses localStorage to track feature usage — only with your explicit consent. You can withdraw consent at any time via the cookie settings banner.

No advertising, retargeting, or third-party tracking cookies are used. Clearing browser cookies alone will not log you out — you must use the logout button to clear your session token from localStorage.

7. Data Retention

  • Account and Profile Data: Retained while your account is active
  • Application Data: Retained until you delete it
  • Server Log Files: Retained for up to 30 days on the instance host
  • Deleted Account Data: Permanently removed immediately upon account deletion

8. Children's Privacy

This software is not intended for use by individuals under 16 years of age. The instance operator is responsible for enforcing appropriate age restrictions.

9. Changes to This Policy

The project maintainers may update this policy in the source repository. Instance operators are responsible for keeping this page current with their own deployment practices.

10. Contact

For questions about your data on this specific instance, contact the person who set it up. For questions about the open-source software itself, open an issue on the GitHub repository.

You can exercise most data rights yourself without contacting anyone: export your data or delete your account directly from Dashboard → Settings.
